What is Handlebars?

Handlebars is a simple templating language.
It uses a template and an input object to generate HTML or other text formats. Handlebars templates look like regular text with embedded Handlebars expressions.

Paginator label

Sets the "d_pagination_label" variable as an array of joined pipe strings corresponding to the buttons First|Previous|Next|Last.
ex: {{setVariable "d_pagination_label" "Prima|Prossima|Precedente|Ultima"}}
To not display a button, omit the string between the pipes.
ex: {{setVariable "d_pagination_label" "|Next|Prev|"}}.
If the variable is not set, the following tuple "First|Previous|Next|Last" will be used.
