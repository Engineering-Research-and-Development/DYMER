Element Public/Private

Add the following attributes to the model element:
dymer-model-element="" dymer-model-visibility="private"
the corresponding value will be visible only to authenticated users.
Instead
dymer-model-element="" dymer-model-visibility="public"
the corresponding value will be visible to all users.
