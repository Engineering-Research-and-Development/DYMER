## Code of Conduct

This project and everyone participating in it is governed by the [DYMER Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code. Please report unacceptable behavior to [project-maintainers@example.com].

## Getting Started

### Issues

- **Search existing issues** before creating a new one (look for closed ones too!)
- **Use issue templates** - they help us help you faster
- **Be descriptive** - include screenshots, logs, and steps to reproduce

### Discussions

Use GitHub Discussions for:
- Questions and help requests
- Ideas and feature proposals
- Show and tell

## How to Contribute

### Reporting Bugs

**Bug reports are welcome!** Please include:

```markdown
**Describe the bug**
A clear description of what the bug is.

**To Reproduce**
Steps to reproduce the behavior:
1. Go to '...'
2. Click on '....'
3. Scroll down to '....'
4. See error

**Expected behavior**
What you expected to happen.

**Screenshots**
If applicable, add screenshots.

**Environment:**
 - OS: [e.g., Ubuntu 22.04]
 - Docker version: [e.g., 24.0.0]
 - DYMER version: [e.g., 3.0.1]
 - Browser: [e.g., Chrome 120]

**Additional context**
Add any other context about the problem here.
```
