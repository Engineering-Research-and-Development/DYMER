Visibility
Restricted: resource visibility restricted only to the current workspace members
Private: visibility restricted to the resource owner
Public: resource is visible also to other platform workspaces that are using the same resource model

Element Public/Private

Add the following attributes to the model element:
dymer-model-element="" dymer-model-visibility="private"
the corresponding value will be visible only to authenticated users.
Instead
dymer-model-element="" dymer-model-visibility="public"
the corresponding value will be visible to all users.

Element Public/Private

Add class needs-validation and attribute novalidate in your model form, if they do not exist 
 <form id="entityForm" class="senderForm  needs-validation" novalidate=""></form>

for each element in you want to display error messages, add this div with class invalid-feedbach 
<div class="invalid-feedback">test invalid</div>

for each element in you want to a custom validate, add attribute dymer-element-validation="your_function_name"
your_function must be return true/false.
Ex 
<textarea dymer-model-element="" dymer-model-visibility="private" dymer-element-validation="testdescr" type="textarea"
            class="form-control  col-12 span12" name="data[description]" minlength="10" maxlength="20"></textarea>

function testdescr(el){
    if(el.val().length >12){
        return true;
    }else{
        return false;
    }
}
