The following is an example of how to setup a simple Dymer Viewer Map&DataTable.

The entities must have a location attribute to be viewed inside the map as a marker! 
    "location": {
      "type": "Point",
      "coordinates": ["16.5058996","48.2256476"]
    }

Configuration

Create a js function named mainDymerMap(), define the json that configures the Viewer and pass it in the funtion generateMapDT()

 <script>
    var jsonDymerConfig = {
        query: { // define the query to do in dymer
            "instance": {
                "index": index-entity  
                "type":  type-entity  
            }
        },
        endpoint: 'entity.search', // set the endpoint to execute the query of entities
        viewtype: 'teaserlist', // set to use the 'Preview in List' template
        swapgeop: true,
        dt: { // set data-table columns
            columns: [
              { title: "Title", data: "title" },
              { title: "", data: "_id", 
                  render: function (data, type, row) {
                      return '<i class="fa fa-eye" aria-hidden="true" style="cursor:pointer;color:#17a2b8" onclick="kmsrenderdetail(\'' + data + '\')"></i>';	},
                className: 'text-center', "orderable": false}		
              ]
      },
       target: {
            fullcontent: {// configure where and how to render the detail of an entity
                id: "#cont-MyEnt", // the Map & DataTablewill be rendered inside the element with id "#cont-MyList"
                action: "html" // set the method to insert the content (html/append/prepend)
                }
            map:{  id: "#cont-TotalMap"// configure id map container (if map/dt view) },
            dt:{  id: "#cont-Dt"// cconfigure id dataTable container (if map/dt view)},
            list:{  id: "#cont-MyList"// configure id list container (if list view)}
            }
        }
    };
    function mainDymerView() {
        generateMapDT(jsonDymerConfig);
    }
 </script>

Import the dymer.map.js with id 'dymerurl' and set the <dymerip> 

 <script id="dymerurl" src="<dymerip>/public/cdn/js/dymer.map.js"> </script>

Create a DOM element where render the Entity details (id="cont-MyEnt" in this case, an element with id="cont-Map" where render the Map and an element with id="cont-Dt" where render the DataTable

 </div>

version 2

Dymer Viewer imports Bootstrap v4 and jQuery 3, if you use previous versions add a json object called dymerconf as below to exclude the import. To set the 'user id' and 'group id' add the variables d_uid and d_gid. 

 var d_uid = '<user-id>'; // user id
 var d_gid = '<group-id>'; // group id 
 var dymerconf = { // disable import bootstrap & jquery
    notImport : ["bootstrap","jquery"]
 };

Data Table

map:{ 
    dt: {
        columns: [{ 
            defaultContent: "",
            className: ' noselectfilter',//onlyDateSearch, noautofilter , tdEllipse
            visible: false
            } ] 
        }
    }

className: noselectfilter = remove select filter in the footer
className: onlyDateSearch = filter for date
className: noautofilter = remove input filter in the head
visible: false = remove the filed form the table and it can be activate form 'Set Field Visibility'

        "zoom": 6.5,
        "maxBounds": [
            [-135, -270],
            [135, 270]
        ]
