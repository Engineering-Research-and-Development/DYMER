Following the code needed to view a light version of DYMER administration system.
With the light version it is possible:

  • Upload and View a Model
  • Upload and View a Template
  • Create new Entity

Create a js function named mainDymerModel(), define the json that configures the query to execute and pass it in the funtion loadFormList(url,target,query)

 <script>
    var d_uid='<user-id>'; 
	var d_gid='<group-id>';
    function mainDymerModel() {
        var myQuery = {};
        
        var geturl = getendpoint('form');  // model's url
        myQuery = { "query": { "instance._index": { $ne: "general" } } };  // all models with index not equal 'general'
        loadFormList(geturl, $('#cont-FormList'), myQuery);
        
        myQuery = { "query": { "instance._index": "general" } }; // all models with index equal 'general'
        loadFormList(geturl, $('#cont-GeneralList'), myQuery);

        geturl = getendpoint('template');  // template's url
        myQuery = { "query": { "instance._index": { $ne: "general" } } }; 
        loadFormList(geturl, $('#cont-TemplateList'), myQuery); 
    }
 </script>

Import the dymer.manager.js with id 'dymerurl' and set the <dymerip> 

 <script id="dymerurl" src="<dymerip>/public/cdn/js/dymer.manager.js"> </script>

Create a DOM element where render the lists of models and templates with the same id passed in loadFormList and one lement with id="cont-RenderForm" where will be rendered the Models/Templates

 <div class="row">
    <div class="col-2">
        <label class="kms-title-label">Uploader</label>
        <div id="cont-GeneralList">
        </div>
        <label class="kms-title-label">Create Entities</label>
        <div id="cont-FormList">
        </div>
        <label class="kms-title-label">View Templates</label>
        <div id="cont-TemplateList">
        </div>
    </div>
    <div id="cont-RenderForm" class="col-10">    
    </div>
 </div>

Dymer Viewer imports Bootstrap v4 and jQuery 3, if you use previous versions add a json object called dymerconf as below to exclude the import. To set the 'user id' and 'group id' add the variables d_uid and d_gid. 

 var d_uid = '<user-id>'; // user id
 var d_gid = '<group-id>'; // group id 
 var dymerconf = { // disable import bootstrap & jquery
    notImport : ["bootstrap","jquery"]
 };
