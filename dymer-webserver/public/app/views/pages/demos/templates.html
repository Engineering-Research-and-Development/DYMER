What is Handlebars?

Handlebars is a simple templating language.
It uses a template and an input object to generate HTML or other text formats. Handlebars templates look like regular text with embedded Handlebars expressions.

Paginator label
sets the "d_pagination_label" variable as an array of joined pipe strings corresponding to the buttons First|Prevous|Next|Last
ex {{setVariable "d_pagination_label" "Prima|Prossima|Precedente|Ultima"}}
to not display the button omit the string between the pipes
{{setVariable "d_pagination_label" "|Next|Prev|"}}.
if the variable is not set, the following tuple "First|Prevous|Next|Last" will be used

Set Variable 

{{setVariable "your_variable_name" your_value}}
Example:
{{setVariable "d_pagination_size" 6}}

If Equals 

{{#ifEquals your_variable_name your_value}}
    your code
{{else}}
    your code
{{/ifEquals}}
Example:
{{#ifEquals @index 0}}HELLO!!{{/ifEquals}}

Render File 

<img class="img-fluid logof" src="{{loadfile _id your_object_key.id}}">
Example:
<img class="img-fluid logof" src="{{loadfile _id logo.id}}">

If the attached file is in a json array structure

{{#each your_object_array_key}}
    <a title="{{label}}" href="{{loadfile relative_path_root_entity/_id your_object_key.id}}" target="_blank">{{label}}</a>
{{/each }}
Example:
{{#each additionalmaterial}}
    <a title="{{label}}" href="{{loadfile ../../_id pdf.id}}" target="_blank">{{label}}</a>
{{/each }}

Format Date 

{{formatDate your_object_key "date format desired"}}
Example:
Updated {{formatDate properties.changed "dd/MM/yyyy HH:mm"}}

If Is Nth Item 

{{#each this}}
    {{#ifIsNthItem nth=your_number_value}}
        yes
    {{else}}
        no
    {{/ifIsNthItem}}
{{/each}}
Example:
{{#each this}}
    {{#ifIsNthItem nth=8}}
        yes
    {{else}}
        no
    {{/ifIsNthItem}}
{{/each}}
